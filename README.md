# ExlibrisProject

# Installation Process.
1. Clone the repo.
2. Open the folder containing the project. 
3. Open terminal in the folder.
4. Use the command "npm install cypress --save-dev" or "yarn add cypress --dev" to install cypress.
5. Use the command "npx cypress open" to launch cypress. 
6. Run the test.

# NB:You need to install nodejs.
